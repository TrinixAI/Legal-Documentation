# Support

**Trinix File Toolbox**

Effective Date: 2026-05-20

## 1. How to Reach Us

- **General support:** <support@trinix.gg>
- **Sales and billing:** <sales@trinix.gg>
- **Website:** <https://trinix.gg>

## 2. What Support Covers

Support is provided for:

- Installation problems on supported versions of Windows.
- Crashes or unexpected errors inside Trinix File Toolbox.
- Conversion failures on a file that other tools can read.
- Microsoft Store license activation issues for users who purchased through
  the Microsoft Store.
- Direct purchase issues for users who bought through <https://trinix.gg>.

## 3. What Support Does Not Cover

- Recovering files damaged before Trinix File Toolbox was used on them.
- Format-specific consulting (for example, designing the layout of a PDF).
- Modifying or extending the Software for a customer's private use.
- Issues caused by user-installed third-party tools such as FFmpeg, except
  to help confirm whether the third-party tool was detected.

## 4. Information to Include in a Support Request

Please include, where applicable:

- A short description of the issue and what you were trying to do.
- The Trinix File Toolbox version (visible in Settings).
- Your Windows version (for example, Windows 11 23H2).
- Steps to reproduce the issue.
- A diagnostics export, if the issue is reproducible. To create one, use
  the **Export Diagnostics** action inside the Software. The export
  contains no personal data and no file contents. Sending it to us is
  optional and is entirely your decision.

## 5. Response Time

We aim to respond to support requests within five (5) business days.
Response time may be longer around major holidays or during product
launches.

## 6. Languages

Support is provided in English.
