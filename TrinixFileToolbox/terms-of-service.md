## 2. Description of Services

Trinix File Toolbox is a local-first Windows desktop utility for converting
and processing common file types. The Software currently includes the
following tools:

- PDF to TXT extraction
- TXT to PDF rendering
- Merge PDFs
- Merge PDFs into a single TXT
- Merge TXT files into a single PDF
- PDF to PNG/JPG rasterization
- Audio conversion (requires a separately installed copy of FFmpeg)
